Ce script formatte la clé USB pour l'utiliser avec le logiciel HP.

Attention : N'avoir qu'une seule clé USB dans votre ordinateur !! La clé sera formatée!

Avant d'utiliser le logiciel, aller dans PowerShell en tant qu'administrateur et entrez : set-executionpolicy RemoteSigned CurrentUser

Fermez la fenêtre et lancez le script avec clic droit -> executer avec PowerShell.

A la fin du script, l'ordinateur vous demandera de formatter la clé, ignorez les fenêtres.

le logiciel HP s'ouvrira, vous pourrez maintenant formater la clé.
